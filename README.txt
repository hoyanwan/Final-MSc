This is a Unity3D project with version 2022.3.5f1


This Unity project use ML-Agents to do machine learning.
The project should be able to run by itself.

If there is any issue with the running the code,
makesure python is install
Then go to the file directory and type in cmd
Next we create a python virtual environment
type python -m venv venv
type vevn\scripts\activate
there should be a (venv) in the start of the line
type python -m pip install --upgrade pip
type pip3 install torch torchvision torchaudio
type pip install protobuf==3.20.3

This should make the project work
If it is still not working, make sure ML Agents in
Package Manager in installed.
